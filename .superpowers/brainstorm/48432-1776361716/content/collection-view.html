Key details

  • Groups sidebar within the main panel — "All saved" at top, user-created groups, "Ungrouped" at bottom. Each shows item count.
  • Post vs comment distinguished by colored badge — blue for posts, purple for comments.
  • Comment titles show "Re: {parent post title}" so you always know the context.
  • Tags from the original P2 post rendered as small chips below the excerpt.
  • Click an item → exits collection view, navigates to the original site/post, opens detail panel.
  • Filter input searches across titles, authors, content, and tags.
  • Drag items between groups to organize (or right-click → Move to group).
